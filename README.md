\## Название проекта

Прогнозирование популярности поста (лайки, репосты) на основе его контента, времени публикации и автора.



\## Краткое описание

Проект выполняет анализ датасета Twitter-постов (1000 записей) и строит модель линейной регрессии для предсказания количества лайков и репостов.
