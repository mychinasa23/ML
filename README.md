# Название проекта

Прогнозирование популярности поста (лайки, репосты) на основе его контента, времени публикации и автора.

---

## Краткое описание

Проект выполняет анализ датасета Twitter-постов (1000 записей) и строит модель линейной регрессии для предсказания количества лайков и репостов на основе текстового контента и временных признаков.

---

## Требования

- Python 3.9+
- pip

---

## Как запустить

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/mychinasa23/ML.git
cd ML
```

## 2. Установите зависимости

```bash
pip install -r requirements.txt
```

## 3. Запустите Jupyter Notebook

```bash
jupyter notebook project.ipynb
```

## Структура файлов

```
ML/
├── project.ipynb           # Jupyter Notebook (обучение, графики, анализ)
├── twitter-posts.csv       # датасет с постами
└── requirements.txt        # зависимости проекта
```

## Рекомендация

Все результаты выполнения кода сохранены в Jupyter Notebook и видны при открытии файла на GitHub. Для повторного запуска необходимо установить зависимости из requirements.txt и запустить ноутбук в Jupyter.
